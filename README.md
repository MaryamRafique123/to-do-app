# to-do-app
📝 To-Do App

A simple, clean, and responsive To-Do List web application built using HTML, CSS, and JavaScript. This app helps users manage their daily tasks efficiently by allowing them to add, delete, and mark tasks as completed.

🚀 Features
➕ Add new tasks
✔️ Mark tasks as completed
❌ Delete individual tasks
🧹 Clear all tasks (if implemented)
💾 Data stored in browser (Local Storage if used)
📱 Fully responsive design (mobile + desktop)


🛠️ Built With
HTML5 – Structure of the app
CSS3 – Styling and layout
JavaScript (Vanilla JS) – Functionality and interactivity
📂 Project Structure


to-do-app/
│
├── index.html        # Main HTML file
├── style.css         # Styling file
├── script.js         # JavaScript logic
└── README.md         # Project documentation
⚙️ How to Run the Project

Download or clone the repository:

git clone https://github.com/MaryamRafique123/todo-app.git
Open the project folder.
Open index.html in your browser.

That’s it — no installation required.

💡 How It Works
Users enter a task in the input field.
Click the Add button to insert it into the list.
Click on a task to mark it as completed.
